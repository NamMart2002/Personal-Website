Career Village - Data Analytics


Description

I spent the summer working as an intern for PricewaterhouseCoopers(PwC). I worked as a project lead for a non-profit named CareerVillage. CareerVillage is a website that helps students looking for advice connect with industry experts to answer their career questions. My team and I analyzed several data sheets and spent time on the website to get a better feel for it. From there we recommended a number of improvements to their user interface, data collection, and data utilization in order to improve their website's user experience.
